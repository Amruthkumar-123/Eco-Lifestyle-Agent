# 🌱 EcoGuide AI – Eco Lifestyle Agent

## 📌 Project Overview

EcoGuide AI is an AI-powered Eco Lifestyle Assistant developed as part of the IBM SkillsBuild AICTE Internship Project.

The chatbot uses the Retrieval-Augmented Generation (RAG) approach to provide reliable and environmentally responsible guidance by retrieving information from uploaded environmental documents.

---

## 🎯 Problem Statement

Problem Statement No. 6 – Eco Lifestyle Agent

Develop an AI assistant that promotes sustainable living by answering user questions using trusted environmental information.

---

## 🚀 Features

- ♻️ Plastic Waste Management Guidance
- 🌿 Sustainable Living Tips
- 💧 Water Conservation Suggestions
- ⚡ Energy Conservation Advice
- 🗑️ Waste Segregation Guidance
- 📚 Retrieval-Augmented Generation (RAG)
- 🤖 AI-powered Conversational Assistant
